## Quick Start

```
/week-ritual                → One ritual suggestion
```

**What you get:** One specific weekly ritual — something small, sustainable, and
tailored to their life. Appended to `outputs/activities/rituals.md`.

---

## What a Good Ritual Is

A ritual is:
- **Recurring** — weekly or a set rhythm (every Sunday, first Saturday of the month, etc.)
- **Small** — 20 minutes to 2 hours, not a production
- **Anchored** — tied to a time or trigger so it's easy to maintain
- **Theirs** — specific enough that it wouldn't work for a generic couple

A ritual is NOT:
- A date idea (those go in /date-idea)
- A one-off activity
- Something that requires planning each time

## Pre-Flight: Context Check

Read:
1. `context-library/preferences.md` — Interests, energy level, morning/evening
2. `context-library/constraints.md` — Work schedule, weekend availability
3. `context-library/memory/learnings.md` — What they enjoy
4. `outputs/activities/rituals.md` — Already suggested rituals (avoid repeating)

## Output Format

```markdown
## [Ritual Title]

**The ritual:** [2-3 sentences. What it is, when it happens, what it looks like in practice.]

**Why it fits you:** [1 sentence. Ties directly to their schedule, preferences, or stage.]

**When to anchor it:** [Specific time suggestion — "Sunday morning before noon,"
"First Friday of the month," "Any weeknight you're both home by 7"]

**What you need:** [Minimal list — or "nothing, just show up"]

**How to start:** [One specific action to do this week to begin]
```

## Save Output

Append to: `outputs/activities/rituals.md` (create if it doesn't exist)
Do not overwrite — this file accumulates all ritual suggestions over time.
